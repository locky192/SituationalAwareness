# SituationalAwareness

This repository stores public SEC 13F filings for Situational Awareness LP and a Next.js dashboard that visualizes the reported positions across all filings.

See [sec-filings/README.md](sec-filings/README.md) for the filing index.

## Run the dashboard

```bash
npm install
npm run dev
```

The data file is generated from the archived SEC XML filings:

```bash
npm run data:build
```
